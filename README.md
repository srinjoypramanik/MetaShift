# MetaShift

## Game Description

**MetaShift** is a 2D arcade-style side-scrolling action game developed using the **iGraphics** library in C. The game focuses on vehicle evolution, terrain-based movement, and strategic resource management where players must balance coins and fuel to progress through levels.

## Features
- Four level-based progression system with increasing difficulty.
- Vehicle evolution system with new abilities unlocked in each level.
- Different enemy types requiring different combat methods.
- Resource management system using coins and fuel.
- Shooting mechanics and final transformation into a combat mech.


## Project Details
IDE: Visual Studio 2013

Language: C, C++

Platform: Windows PC

Genre: 2D arcade action


## How to Run the Project

Make sure you have the following installed:
- **Visual Studio 2013**
- **MinGW Compiler** (if needed)
- **iGraphics Library** (included in the project)

Open the project in Visual Studio 2013:
- Open Visual Studio 2013
- Go to File → Open → Project/Solution
- Select the .sln file
- Click Build → Build Solution
- Run using Debug → Start Without Debugging

How to Play

- Boost             |   B 
- Shoot	            |  Space
- Pause             |  	P


### **Game Rules**
- The player must collect required coins in each level to unlock the next level.
- Coins have fixed limits per level; extra coins give no advantage.
- Fuel is required for shooting and survival in later levels.
- Different enemies require different methods to defeat (collision or shooting).
- Colliding with restricted enemies results in game over.
- Progress depends on decisions made in earlier levels.
